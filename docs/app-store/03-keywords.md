## 选词逻辑

- **横琴 + 驻留 + 天数 + 183 + 124** 锁定核心场景；这些词搜索量小但意图非常匹配，转化率高
- **菜单栏 + 热力图** 锁定 Mac 工具偏好的高端用户
- **iCloud** 帮助和"无账号同步"诉求的用户对齐
- 没放任何竞品名（Apple 禁止）
- 没放无关流行词（如"高效"、"AI"），降低被算法压低权重的概率

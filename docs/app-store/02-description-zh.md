## 简介（前 4 行优先）

一年几天，是一个驻留在 macOS 菜单栏的"地点驻留追踪"小工具。

每一年，你想在某个地方住够多少天？183 天的居住认证、124 天的工作日补贴、思乡的家人、远行的远方——它们都需要数清楚一整年里你"真的待过"的日子。一年几天把这件事变成菜单栏里一个看一眼就懂的色块图。

## 主要特性

- **菜单栏一眼看数** — 当前在追踪地点累计了多少天，183 / 124 的进度长什么样，菜单栏标题直接告诉你
- **多坐标档案** — 横琴、深圳、老家、海外，每个地点一份独立数据，自由切换
- **年度热力图 / 月度日历** — 同一份数据两种看法，绿色是 GPS 自动检测到、深绿是手动标记、蓝色是请假桥接、黄色是节假日桥接
- **法定假期自动桥接** — 中国 2026 年全套节假日和调休内置，国庆前后都在追踪地点就自动算上整段假期
- **跨设备 iCloud 同步** — 系统级 Key-Value Storage，登录同一个 Apple ID 自动合并，不需要任何账号体系
- **导入导出 JSON** — 与 Chrome 扩展版同格式互通，搬迁数据无门槛
- **本地优先 · 零追踪** — 所有数据存在你的 Mac 和你的 iCloud 里，没有任何第三方服务器，永远没有广告

## 业务场景

- 横琴粤澳深度合作区个税补贴：每年 183 自然日 / 124 工作日驻留判定
- 自由工作者：在不同城市的实际居住时长统计
- 跨城工作者：周末家、工作日单位的天数比例
- 出差党：一年到底在外地呆了多久

## 设计哲学

- 不打扰：菜单栏 24×24 像素的色块就是全部，按需点开看详细
- 不滥用权限：定位仅在你按下"使用当前位置"时请求一次
- 不索取付费：基础功能完全免费，没有内购解锁项

## 关于横琴

应用最初为珠海横琴粤澳深度合作区的 183 自然日 / 124 工作日居住认证而生（这也是默认坐标档案 22.125°N, 113.535°E, 8km 半径的来源），但可以追踪任何你想关心的地点。

## 隐私

- 不收集任何个人信息
- 定位数据仅在本机使用，从不上传
- 跨设备同步走 Apple iCloud Key-Value Storage，数据加密驻留你自己的 iCloud
- 没有自有服务器、没有第三方分析 SDK
